# Internship Management Dashboard

## Project Overview

This project is a customized version of the Mazer Bootstrap 5 Admin Dashboard template.  
The original template was modified and redesigned into an **Internship Management Dashboard** as part of the Front-End Skill Assessment.

The goal of this task was to demonstrate the ability to work with an existing codebase, customize UI components, modify layouts, and create a clean professional dashboard interface.

---

## Template Used

- Mazer Admin Dashboard
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- Vite

---

## Changes Made

### 1. Dashboard Theme Changed

The original profile/statistics dashboard was redesigned into an **Internship Management Dashboard**.

Updated page heading:

- Profile Statistics → Internship Management Dashboard

---

### 2. Dashboard Cards Updated

The original cards were replaced with internship-related metrics:

- Total Interns
- Completed Tasks
- Pending Reviews
- Active Projects

All cards were aligned and adjusted for better spacing and consistent height.

---

### 3. Sidebar Branding Updated

The default template branding text was replaced with custom project branding.

- Mazer → Internship Admin

---

### 4. Sidebar Navigation Customized

Unnecessary demo menu items were removed or planned for cleanup.

Navigation was adjusted to suit internship dashboard structure such as:

- Dashboard
- Interns
- Tasks
- Reports
- Settings

---

### 5. Chart Section Modified

The original analytics chart was reused and renamed as:

- Monthly Task Completion Report

Chart section now represents internship task progress data.

---

### 6. Department Performance Section Added

The profile region section was modified into team performance analytics:

- Backend Team – 88%

### 8. Right Panel Updated
